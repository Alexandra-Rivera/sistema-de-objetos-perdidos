# Proyecto SOP (Sistema de Objetos Perdidos) de la UEES

Este proyecto tiene la finalidad de gestionar las entradas y salidad de objetos pérdidos encontrados dentro de la universidad en las instalaciones correspondientes de la facultad de ingeniería. Se busca cumplir con el objetivo de:
- Ser una herramienta eficaz para verificar rápidamente si algún objeto está o no dentro del inventario.
- Mantener un registro de aquellos objetos que ya fueron extraídos.
